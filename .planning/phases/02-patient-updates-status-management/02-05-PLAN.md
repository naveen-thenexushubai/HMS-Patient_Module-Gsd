**Helper method for patient registration (reuse in multiple tests):**

```java
private UUID registerTestPatient(String firstName, String lastName) throws Exception {
    String requestBody = """
        {
          "firstName": "%s",
          "lastName": "%s",
          "dateOfBirth": "1990-05-15",
          "gender": "MALE",
          "phoneNumber": "+12345678901",
          "photoIdVerified": true
        }
        """.formatted(firstName, lastName);

    String responseJson = mockMvc.perform(post("/api/v1/patients")
            .contentType(MediaType.APPLICATION_JSON)
            .content(requestBody)
            .with(user("receptionist").roles("RECEPTIONIST")))
        .andExpect(status().isCreated())
        .andReturn()
        .getResponse()
        .getContentAsString();

    return UUID.fromString(JsonPath.read(responseJson, "$.businessId"));
}
```
